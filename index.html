Run It Locally

This project is built with Tkinter and OpenCV, so it needs local access to your webcam. Vercel cannot run the desktop GUI directly.

git clone https://github.com/JuniorRizzler/Quantum-RNG.git cd Quantum-RNG python -m pip install -r requirements.txt python app.py

Why This Project Is Different

Physical entropy source

Instead of using a normal pseudo-random function directly, the app samples real camera sensor noise and frame-to-frame changes.

Multiple extraction pipelines

Compare raw least-significant bits, Von Neumann bias reduction, and SHA-256 whitened output.

Experiment logging

Record different conditions like lens covered, room light, and moving scene, then compare their output quality.

Report export

Export JSON and Markdown reports for school demos, judging, or GitHub examples.
