Method

  1. Capture webcam frames and convert them to grayscale.
  2. Compute frame-to-frame pixel differences to emphasize sensor noise and motion.
  3. Extract least-significant bits from the mixed pixels.
  4. Optionally reduce bias with Von Neumann extraction.
  5. Optionally whiten with SHA-256 using the browser crypto API.
  6. Run statistical checks, generate integers, and export a report.

This is a browser-based hardware-noise RNG prototype, not a certified true quantum random number generator.
